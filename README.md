# visitant.vim

minimal greyscale theme for Vim.

### screenshots

![screenshot1](./assets/screenshot1.png)
![screenshot2](./assets/screenshot2.png)

### installation

with packer:

```
return require('packer').startup(function(use)

use { 'timmyha/visitant.vim' }

end)
```

### usage

`:colorscheme visitant`
